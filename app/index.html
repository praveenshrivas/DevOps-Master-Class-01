Python Day 01 – Variables 🐍
| Store and reuse values in Python
What is a Variable?

A variable is a container used to store data. In simple words, a variable is a name that holds a value.

Example
name = "Praveen"
Why Variables?
  • Reusable – Store values once and use them again and again
  • Clean code – Makes your script easier to read and manage
  • Easy updates – Change the value in one place only
  • Python is smart – No need to define the data type manually
Naming Rules & Best Practice
  • Allowed: my_name = "Praveen", age1 = 25
  • Not allowed: 1name, my-name
  • Best practice: Use snake_case like my_variable_name
simple readable snake_case
Example Code
>>>name = "Your Name"
>>>experience = 9
>>>role = "DevOps Engineer"
>>>print(name)
>>>print(experience)
>>>print(role)
Important Tip
👉 Variables can be changed anytime in Python
👉 Example: x = 10 then x = 20
DevOps Use Case
env = "production"
region = "ap-south-1"
instance_type = "t2.micro"
  • Used in deployment scripts
  • Helpful in automation
  • Important for configuration management
One-Line Summary
👉 Variable is a container to store the data.
